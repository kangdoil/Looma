# Looma MVP PRD

## 🎯 핵심 정보

**목적**: IT 기획자·디자이너가 업무 중 발생한 문제를 이미지 위에 직접 주석으로 기록하여 개인 노하우를 시각적 지식 자산으로 축적할 수 있게 한다
**사용자**: 업무 중 시각적 자료를 자주 다루며 문제 해결 과정을 체계적으로 관리하고 싶은 IT 기획자, UI/UX 디자이너, 개발자

---

## 🚶 사용자 여정

```
1. 홈 페이지 (기록 생성 진입점)
   ↓ 기록 유형 선택

   [이미지 기록 선택] → 이미지 선택 페이지 (Step 1)
   [텍스트 기록 선택] → 정보 입력 페이지 (Step 3, 이미지 없이)

2. 이미지 선택 페이지 (Step 1/3)
   ↓ 갤러리에서 이미지 선택

3. 주석 작성 페이지 (Step 2/3)
   ↓ 이미지 위 포인트 탭 → 주석 내용 입력 → 완료

4. 정보 입력 페이지 (Step 3/3)
   ↓ 제목·태그 설정 → 저장

   저장 성공 → 홈 페이지 복귀
   저장 실패 → 에러 표시 (정보 입력 페이지 유지)

━━━━━━━━━━━━━━ 하단 탭 전환 ━━━━━━━━━━━━━━

5. 라이브러리 페이지 (하단 탭: 라이브러리)
   ↓ 칩 태그 필터 / 키워드 검색 → 기록 카드 클릭

6. 기록 상세 페이지
   ↓ 이미지 위 포인트 탭 → 바텀시트 확장 → 주석 상세 조회
   → 뒤로가기 → 라이브러리 페이지
```

## ⚡ 기능 명세

### 1. MVP 핵심 기능

| ID       | 기능명               | 설명                                                                       | MVP 필수 이유                | 관련 페이지                        |
| -------- | -------------------- | -------------------------------------------------------------------------- | ---------------------------- | ---------------------------------- |
| **F001** | 이미지 선택          | 기기 갤러리에서 이미지 선택 또는 카메라 촬영                               | 시각적 기록의 핵심 입력 수단 | 이미지 선택 페이지                 |
| **F002** | 이미지 주석 추가     | 이미지 위 임의의 지점 탭 시 포인트 핀 생성 및 문제점·해결 과정 텍스트 입력 | 서비스 핵심 차별화 기능      | 주석 작성 페이지, 기록 상세 페이지 |
| **F003** | 주석 시각화          | 포인트 핀을 이미지 위에 표시, 탭 시 바텀시트에서 내용 확인·수정·삭제       | 기록의 시각적 명확성 보장    | 주석 작성 페이지, 기록 상세 페이지 |
| **F004** | 텍스트 기록          | 이미지 없이 텍스트만으로 문제·해결 과정 작성                               | 이미지 없는 상황의 기록 지원 | 정보 입력 페이지                   |
| **F005** | 기록 메타데이터 입력 | 제목·태그 설정 후 기록 저장                                                | 라이브러리 분류·검색의 기반  | 정보 입력 페이지                   |
| **F006** | 라이브러리 목록 조회 | 저장된 기록을 2열 카드 그리드로 조회                                       | 기록 재활용의 진입점         | 라이브러리 페이지                  |
| **F007** | 검색 및 태그 필터    | 키워드 검색, 칩 태그 필터로 기록 탐색                                      | 지식 재활용 효율화           | 라이브러리 페이지                  |
| **F008** | 기록 상세 조회       | 이미지+포인트 조회, 바텀시트 확장으로 주석 상세 및 태그 요약 확인          | 과거 기록 재활용의 핵심      | 기록 상세 페이지                   |

### 2. MVP 필수 지원 기능

| ID       | 기능명    | 설명                                  | MVP 필수 이유                   | 관련 페이지                    |
| -------- | --------- | ------------------------------------- | ------------------------------- | ------------------------------ |
| **F010** | 기본 인증 | 이메일 회원가입 / 로그인 / 로그아웃만 | 개인 기록 보호를 위한 최소 인증 | 로그인 페이지, 회원가입 페이지 |

### 3. MVP 이후 기능 (제외)

- 프로필 상세 관리 (아바타, 자기소개 등)
- 카테고리 계층 관리 (현재는 태그로 단순화)
- 소셜 공유 및 팀 협업
- 실시간 알림 시스템
- AI 기반 자동 태그 추천

### 로그인 페이지

> **구현 기능:** `F010` | **인증:** 비로그인 전용

| 항목            | 내용                                                                                           |
| --------------- | ---------------------------------------------------------------------------------------------- |
| **역할**        | 기존 사용자 인증 전용 페이지                                                                   |
| **진입 경로**   | 앱 최초 진입 시, 또는 인증 만료 시 자동 리디렉션                                               |
| **사용자 행동** | 이메일과 비밀번호 입력 후 로그인, 또는 회원가입 페이지로 이동                                  |
| **주요 기능**   | • 이메일·비밀번호 입력 폼<br>• 입력값 유효성 검사<br>• 비밀번호 찾기 링크<br>• **로그인** 버튼 |
| **다음 이동**   | 성공 → 홈 페이지, 실패 → 에러 메시지 표시                                                      |

| **진입 경로**   | 로그인 성공 후 자동 이동, 하단 탭의 홈 아이콘 탭                                                                                                     |
| **사용자 행동** | 날짜 확인 후 이미지 기록 또는 텍스트 기록 옵션 카드 선택                                                                                             |

| **다음 이동**   | 이미지 기록 선택 → 이미지 선택 페이지, 텍스트 기록 선택 → 정보 입력 페이지                                                                           |

### 이미지 선택 페이지

> **구현 기능:** `F001` | **인증:** 로그인 필수 | **단계:** Step 1/3

| 항목            | 내용                                                                                                                                                     |
| --------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **역할**        | 기록에 사용할 이미지를 선택하는 첫 번째 단계 페이지                                                                                                      |
| **진입 경로**   | 홈 페이지에서 이미지 기반 기록 선택                                                                                                                      |
| **사용자 행동** | 기기 갤러리 탐색 후 이미지 선택, 또는 카메라 촬영                                                                                                        |
| **주요 기능**   | • 기기 갤러리 그리드 표시<br>• 이미지 선택 시 상단 미리보기 표시<br>• 카메라 즉시 촬영 옵션<br>• 상단: 뒤로가기, 제목, **다음** 버튼, 3단계 진행 점 표시 |
| **다음 이동**   | 이미지 선택 완료 → 주석 작성 페이지                                                                                                                      |

### 주석 작성 페이지

> **구현 기능:** `F002`, `F003` | **인증:** 로그인 필수 | **단계:** Step 2/3

| 항목            | 내용                                                                                                                                                                                                                                                                                                                                                                       |
| --------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **역할**        | 선택한 이미지 위에 포인트 핀을 추가하고 문제·해결 과정을 텍스트로 작성하는 핵심 페이지                                                                                                                                                                                                                                                                                     |
| **진입 경로**   | 이미지 선택 페이지에서 이미지 선택 후 자동 이동                                                                                                                                                                                                                                                                                                                            |
| **사용자 행동** | 이미지 위 특정 지점 탭 → 포인트 핀 생성 → 바텀시트 열림 → 문제점·해결 과정 텍스트 입력 → 포인트 추가 반복 → 완료                                                                                                                                                                                                                                                           |
| **주요 기능**   | • 이미지 전체화면 표시<br>• 이미지 임의 지점 탭 시 포인트 핀(`+` 아이콘) 생성 (F002)<br>• 포인트 생성과 동시에 바텀시트 슬라이드업: 문제점/해결과정 텍스트영역 2개 입력 (F002)<br>• 이미지 위 모든 포인트 핀 시각적 표시, 기존 핀 탭 시 해당 주석 내용 바텀시트에 표시 (F003)<br>• 주석 편집·삭제 기능 (F003)<br>• 상단: 뒤로가기, 제목, **완료** 버튼, 3단계 진행 점 표시 |
| **다음 이동**   | 완료 버튼 클릭 → 정보 입력 페이지                                                                                                                                                                                                                                                                                                                                          |

### 정보 입력 페이지

> **구현 기능:** `F004`, `F005` | **인증:** 로그인 필수 | **단계:** Step 3/3

| 항목            | 내용                                                                                                                                                                                                                                                                |
| --------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **역할**        | 작성한 기록에 제목·태그·설명을 추가하고 최종 저장하는 마지막 단계 페이지                                                                                                                                                                                            |
| **진입 경로**   | 주석 작성 페이지 완료 후 이동 (이미지 기록), 또는 홈에서 텍스트 기록 선택 (텍스트 기록)                                                                                                                                                                             |
| **사용자 행동** | 완성된 이미지+주석 썸네일 확인 → 태그 검색·선택 → 제목·설명 입력 → 저장                                                                                                                                                                                             |
| **주요 기능**   | • 완성된 이미지+포인트 핀 썸네일 표시 (이미지 기록 시), 또는 텍스트 입력 영역 (텍스트 기록 시) (F004)<br>• 태그 검색바 + 기존 태그 칩 선택, 신규 태그 입력 (F005)<br>• 제목·설명 텍스트 입력 폼 (F005)<br>• 상단: 뒤로가기, 제목, **저장** 버튼, 3단계 진행 점 표시 |
| **다음 이동**   | 저장 성공 → 홈 페이지, 저장 실패 → 에러 메시지 표시                                                                                                                                                                                                                 |

| email      | 이메일 (로그인 ID) | string    |
| name       | 사용자 이름        | string    |
| created_at | 가입일             | timestamp |

### records (기록)

| 필드        | 설명                               | 타입/관계      |
| ----------- | ---------------------------------- | -------------- |
| id          | 고유 식별자                        | UUID           |
| user_id     | 작성자                             | → users.id     |
| title       | 기록 제목                          | string         |
| image_url   | 업로드된 이미지 경로 (없으면 null) | string \| null |
| description | 전체 설명 텍스트                   | text           |
| created_at  | 생성일                             | timestamp      |

### annotations (주석)

| 필드       | 설명                        | 타입/관계    |
| ---------- | --------------------------- | ------------ |
| id         | 고유 식별자                 | UUID         |
| record_id  | 소속 기록                   | → records.id |
| x_ratio    | 이미지 내 X 좌표 비율 (0~1) | float        |
| y_ratio    | 이미지 내 Y 좌표 비율 (0~1) | float        |
| problem    | 문제점 텍스트               | text         |
| solution   | 해결 과정 텍스트            | text         |
| created_at | 생성일                      | timestamp    |

| name    | 태그명      | string     |
